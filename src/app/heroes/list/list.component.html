*ngFor/*ngIf Directive & ng-template Element

This exercise show how you can use an *ngFor directive in html template for show the content in an array

 *ngFor="let item of list; let i = index"

Thanks to one way data binding you can see the change in real times, and thanks to *ngIf directive you can hide controls if you don't need them

In this section you can see an ng-template element which shows a message "No hero has been deleted" when the *ngIf condition isn't true"
